# 🏥 Medical Chatbot powered by BERT

A medical question-answering chatbot fine-tuned on the 
[Comprehensive Medical Q&A Dataset](https://www.kaggle.com/datasets/thedevastator/comprehensive-medical-q-a-dataset).

## 📌 Project Info
- **Model:** bert-base-uncased (fine-tuned)
- **Task:** Medical Q&A Classification  
- **Framework:** HuggingFace Transformers + Streamlit
- **Accuracy:** 68.32% (20-class classification)
- **Course:** Natural Language Processing — Universitas Tarumanagara

## ⚠️ Disclaimer
This chatbot is for **educational purposes only**.
Always consult a qualified medical professional for health advice.
